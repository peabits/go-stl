
# io

## 概述

io 包提供了 I/O 原语的基本接口。它的主要工作是将此类原语的现有实现（例如 os 包中的那些实现）包装到抽象功能的共享公共接口以及一些其他相关原语中。

因为这些接口和原语用各种实现包装了较低级别的操作，除非另有通知，否则客户端不应假定它们可以安全地并行执行。
